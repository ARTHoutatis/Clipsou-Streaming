Politique de confidentialité

Dernière mise à jour : 1 novembre 2025

Introduction

Bienvenue sur Clipsou Streaming (ci-après « nous », « notre » ou « le Site »). Nous accordons une grande importance à la protection de votre vie privée et de vos données personnelles.

Cette politique de confidentialité explique quelles informations nous collectons, comment nous les utilisons, les protégeons et les partageons lorsque vous utilisez notre site web clipsoustreaming.com.

Informations que nous collectons

1.1 Informations que vous nous fournissez

Lorsque vous utilisez notre service de soumission de films (Beta), nous pouvons collecter :

  • Informations de connexion Google : nom, adresse e-mail, photo de profil (via Google OAuth 2.0)
  • Informations de chaîne YouTube : nom de la chaîne, ID de la chaîne, URL personnalisée et miniature publique
  • Contenu soumis : titre du film, description, genres, lien YouTube, informations sur les acteurs

Nous ne collectons pas de statistiques de chaîne (comme le nombre d'abonnés) ni d'autres données Google supplémentaires.

1.2 Informations collectées automatiquement

Nous ne recourons à aucun outil d'analyse ou de suivi publicitaire. Les seules données collectées automatiquement proviennent de :

  • Journaux techniques de l'hébergeur : Cloudflare peut journaliser temporairement des informations limitées (adresse IP tronquée, type de navigateur) pour assurer la sécurité et la stabilité du site.
  • Métriques internes agrégées : nous suivons le volume global de demandes ou d'erreurs sans associer ces données à des profils individuels.

1.3 Données stockées localement

Nous utilisons le localStorage de votre navigateur pour stocker :

  • Vos informations d'authentification Google (jeton d'accès, profil de base et détails essentiels de votre chaîne) pour maintenir votre session
  • La progression de vos formulaires (brouillons, images téléversées, étape en cours)
  • Les données anti-abus (horodatages de demande, identifiant de navigateur anonymisé) qui imposent la limite d'une demande par 24 heures
  • Vos préférences (acceptation des conditions, choix du mode OAuth en développement local)

Important : Ces données restent sur votre appareil et ne sont jamais transmises à nos serveurs sans votre consentement explicite.

Comment nous utilisons vos informations

Nous utilisons les informations collectées pour :

  • Fournir et améliorer nos services : streaming de contenu, gestion des demandes de films
  • Authentifier les utilisateurs : via Google OAuth pour sécuriser l'accès à certaines fonctionnalités
  • Modérer le contenu : vérifier et approuver les soumissions de films
  • Communiquer avec vous : répondre à vos questions et demandes
  • Analyser l'utilisation : comprendre comment les utilisateurs interagissent avec notre site
  • Prévenir les abus : détecter et empêcher les activités frauduleuses ou malveillantes

Partage de vos informations

Nous ne vendons jamais vos données personnelles. Nous pouvons partager vos informations uniquement dans les cas suivants :

  • Avec votre consentement : lorsque vous autorisez explicitement le partage
  • Fournisseurs de services :
    • Google (authentification OAuth)
    • YouTube (intégration de vidéos)
    • Cloudflare (hébergement et CDN)
    • GitHub (hébergement du code et des données)
  • Obligations légales : si requis par la loi ou en réponse à une procédure judiciaire
  • Protection des droits : pour protéger nos droits, notre propriété ou notre sécurité

Sécurité de vos données

Nous mettons en œuvre des mesures de sécurité techniques et organisationnelles pour protéger vos données :

  • Chiffrement HTTPS : toutes les communications sont chiffrées
  • OAuth 2.0 : authentification sécurisée via Google
  • Stockage sécurisé : données hébergées sur des serveurs sécurisés (GitHub, Cloudflare)
  • Accès restreint : seuls les administrateurs autorisés peuvent accéder aux données sensibles
  • Surveillance : détection et prévention des accès non autorisés

Note : Aucune méthode de transmission sur Internet n'est 100% sécurisée. Nous nous efforçons de protéger vos données, mais ne pouvons garantir une sécurité absolue.

Stockage local uniquement

Nous n'utilisons aucun cookie sur clipsoustreaming.com. Toutes les informations nécessaires à votre navigation sont stockées localement via le localStorage de votre navigateur (session OAuth, brouillons de formulaire, préférences et protection anti-abus).

Vous pouvez à tout moment vider ce stockage local depuis les paramètres de votre navigateur, ce qui aura pour effet de réinitialiser vos brouillons et de vous déconnecter.

Vos droits

Conformément au RGPD (Règlement Général sur la Protection des Données), vous disposez des droits suivants :

  • Droit d'accès : obtenir une copie de vos données personnelles
  • Droit de rectification : corriger vos données inexactes ou incomplètes
  • Droit à l'effacement : demander la suppression de vos données
  • Droit à la limitation : restreindre le traitement de vos données
  • Droit à la portabilité : recevoir vos données dans un format structuré
  • Droit d'opposition : vous opposer au traitement de vos données
  • Droit de retirer votre consentement : à tout moment

Pour exercer ces droits, contactez-nous à l'adresse indiquée ci-dessous.

Conservation des données

Nous conservons vos données personnelles uniquement aussi longtemps que nécessaire pour :

  • Fournir nos services
  • Respecter nos obligations légales
  • Résoudre les litiges
  • Faire respecter nos accords

Les données d'authentification sont conservées tant que votre compte est actif. Les demandes de films sont archivées après traitement.

Services tiers

Notre site utilise des services tiers qui ont leurs propres politiques de confidentialité :

  • Google OAuth : Politique de Google
  • YouTube : Conditions YouTube
  • Cloudflare : Politique Cloudflare

Nous vous encourageons à consulter ces politiques pour comprendre comment ces services traitent vos données.

Protection des mineurs

Notre site est accessible à tous, mais nous ne collectons pas sciemment d'informations personnelles auprès d'enfants de moins de 13 ans sans le consentement parental.

Si vous êtes parent et pensez que votre enfant nous a fourni des informations personnelles, veuillez nous contacter pour que nous puissions supprimer ces données.

Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité de temps à autre. Les modifications seront publiées sur cette page avec une nouvelle date de « Dernière mise à jour ».

Nous vous encourageons à consulter régulièrement cette page pour rester informé de nos pratiques en matière de confidentialité.
